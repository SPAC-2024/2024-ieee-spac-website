## Getting Started

Clone the repository:

```bash

git clone https://github.com/SPAC-2024/2024-ieee-spac-website.git

```

Install packages:

```bash

npm i
# or
pnpm i
# or
yarn i

```

Then, run the development server:

```bash
npm run dev:next
# or
yarn dev:next
# or
pnpm dev:next
# or
bun dev:next
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
